# 考勤数据看板

基于 ECharts 的 GEU 海外劳务工线上线下考勤比对看板。

## 功能

- 📁 **导入Excel**：维护者上传考勤Excel，自动解析数据
- 📊 **KPI指标卡**：录入率、准确率、补签率、工时异常数
- 🔄 **周期对比**：选择两个周期进行环比分析
- 📈 **图表可视化**：录入率/准确率/补签率对比、打卡数据、补签数据
- 🔍 **明细钻取**：点击表格行查看明细数据
- 📋 **说明备注**：维护者可以添加每期说明

## 使用流程

### 维护者（数据更新）
1. 打开链接 → 点击 **"🔒 编辑模式"** → 输入密码 `admin888`
2. 点击 **"📁 导入Excel"** 上传考勤文件（xlsx格式）
3. 数据自动保存到浏览器
4. 如需导出完整数据包，点击 **"📦 导出数据包"**

### 查看者
打开链接即可查看已嵌入的演示数据，可选择不同周期查看

## 部署说明

本项目已包含全部代码和数据，双击 `index.html` 即可在本地使用。

如需部署到线上，将 `index.html` 上传到任意静态托管服务即可：
- GitHub Pages
- 腾讯云 COS + CDN
- 任意 Web 服务器

> 注意：部署到线上后，维护者导入的数据保存在浏览器 localStorage 中，仅当前设备可见。
> 如需跨设备共享数据，请使用"导出数据包"功能导出 .json 文件分享。
